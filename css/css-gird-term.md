# 📄 CSS -Gird Term

## 1. Grid Layout

### \(1\). Grid Layout  정의

`display:grid;` 를 설정하면, **grid container**가 되고 내부에 있는 자식은  **gird  item**이 되며 격자의 공간 형성하면서 각각의 라인 순서를  가진다.

### \(2\). Gird 용어 \(Term\)

| property | description |
| :--- | :--- |
| 행 \(rows\) |  라인 \(lines\)를 의미하며 수평선 |
| 열 \(coulms\) | 수직 |
